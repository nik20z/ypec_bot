# ypec_bot
## Python 3.8.5


[Бот](https://t.me/ypec_bot "YPEC") предназначен для парсинга расписания с [сайта колледжа](https://www.ypec.ru/rasp-zmnext "Замены на зватра") и рассылки его студентам и преподавателям.
Бот через определённые интервалы времени проверяет наличие замен на завтрашний день и при необходимости производит его рассылку. Также стоит учесть, что в течение дня расписание корректируется, бот отслеживает эти изменения и присылает пользователю обновленную версию

Для взаимодействия с телеграмом используется библиотека aiogram 
В качестве базы данных - встроенная в python sqlite3

* 🔔 __Рассылка__ - подписка на ежедневную рассылку расписания
* 📌 __Закреплять__ - закреплять расписание в чате после рассылки (для бесед потребуется админка)
* ℹ __Заголовок__ - указывать в сообщении для какой группы (преподавателя) было составлено расписание
* 🏷 __Подробно__ - для студентов: отмечать в скобках преподавателей, у которых пройдут занятия. Для преподавателей: отмечать группы
* ⏰ __Время__ - показывать время начала и окончания занятий

5. Дополнительный функционал:
* __Очистка чата__. Реализована таким образом, что в диалоге остаются только сообщения с расписанием, предупреждения о спаме
* __Защита от спамеров__. Бот рассчитывает вреднее время ответа и сравнивает его с заданным порогом. Первый раз пользователь получает предупреждение, но при повторном нарушении блокируется. Время каждой последующей блокировки увеличивается
