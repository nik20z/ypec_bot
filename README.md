# ypec_bot
## Python 3.8.5




[Ссылка на Бота](https://t.me/ypec_bot "YPEC")

[Поддержать проект](https://www.tinkoff.ru/collectmoney/crowd/smirnov.nikita516/vZtFs92623 "Поддержка проекта ypec_bot")
 
Телеграм-бот колледжа ЯПЭК. Предназначен для парсинга расписания с [сайте](https://www.ypec.ru/rasp-zmnext "Замены на зватра") и рассылки его студентам и преподавателям.
Бот через определённые интервалы времени проверяет наличие нового расписания и производит его рассылку. Также стоит учесть, что в течение дня расписание корректируется, бот отслеживает эти изменения и при необходимости повторно производит рассылку

Для взаимодействия с телеграмом используется библиотека aiogram 
В качестве базы данных - встроенная в python sqlite3

2. Меню настроек

![image](https://user-images.githubusercontent.com/62090150/140516467-4b37086e-c815-4f3f-a1e5-e25219aa2383.png)


* __Рассылка__ - подписка на ежедневную рассылку расписания
* __Закреплять__ - закреплять расписание в чате после рассылки (для бесед потребуется админка)
* __Заголовок__ - указывать в сообщении для какой группы (преподавателя) было составлено расписание
* __Подробно__ - для студентов: отмечать в скобках преподавателей, у которых пройдут занятия. Для преподавателей: отмечать группы
* __Время__ - показывать время начала и окончания занятий

5. Дополнительный функционал:
* __Очистка чата__. Реализована таким образом, что в диалоге остаются только сообщения с расписанием и предупреждения о спаме
* __Защита от спамеров__. Бот рассчитывает интервалы между последними запросами и сравнивает среднее время с заданным порогом. Первый раз пользователь получает предупреждение, но при повторном нарушении блокируется. Время каждой последующей блокировки увеличивается
